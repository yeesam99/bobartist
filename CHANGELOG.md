# BobArtist v0.0.38

## 변경 사항
- 게임 제한 시간을 5분으로 추가했습니다.
- 제한 시간이 끝나면 자동으로 RESULT 단계로 이동하고 도망자 승리로 처리합니다.
- 최대 인원을 8명으로 확장했습니다.
- 술래는 1명, 나머지 참가자는 도망자로 배정됩니다.
- 게임 화면에 남은 시간 표시를 추가했습니다.

# BobArtist v0.0.37

## Circle Render Engine
- 술래 화면에서는 별도 원판/base circle을 그리지 않도록 변경했습니다.
- 도망자 paintCanvas를 기본 흰색 배경으로 초기화해, 아무것도 칠하지 않아도 제출 이미지 자체가 흰 원으로 보이게 했습니다.
- 원 테두리/점선/halo가 생기는 렌더링 경로를 줄였습니다.

# BobArtist v0.0.36

## 수정
- 술래 화면에서 v0.0.35의 페더 마스크 때문에 생기던 흰 링/테두리 현상을 제거했습니다.
- 술래 REVEAL/FIND 화면에서는 흰 원판과 내부 그림만 그리고, 외곽선/점선/선택선을 그리지 않도록 고정했습니다.
- 도망자/결과 화면의 기존 가이드 표시 흐름은 유지했습니다.

## 확인
- `npm run build --prefix client` 성공
