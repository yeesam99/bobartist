# v0.0.43 Auto Submit Grace Fix

- 도망자의 제출 버튼을 상단 고정 HUD 우선 영역으로 이동했습니다.
- 그리기 시간이 끝나면 도망자 클라이언트가 현재 원 위치/그림을 자동 제출하도록 보강했습니다.
- 상단 액션 버튼이 좁은 화면에서도 가로 버튼 그룹으로 유지되도록 스타일을 정리했습니다.

# BobArtist v0.0.43

## 변경 사항
- 그리기 제한 시간을 1분으로 분리했습니다.
- 찾기 제한 시간은 5분으로 유지했습니다.
- 시간 초과 시 그리기 단계는 자동 제출/공개 흐름으로 넘어갑니다.
- 게임 상단에 고정 HUD를 추가했습니다.
- 상단 고정 HUD에 방 코드, 남은 시간, 술래/내 점수를 표시합니다.

# BobArtist v0.0.39

## Runner Catch Flow
- 술래가 도망자 1명을 잡아도 게임이 즉시 종료되지 않도록 수정했습니다.
- 잡힌 도망자만 X 표식으로 표시됩니다.
- 모든 도망자를 잡아야 술래 승리로 RESULT 단계에 진입합니다.
- 잡힌 도망자는 다시 선택되지 않도록 처리했습니다.

# BobArtist v0.0.38

## 변경 사항
- 게임 제한 시간을 5분으로 추가했습니다.
- 제한 시간이 끝나면 자동으로 RESULT 단계로 이동하고 도망자 승리로 처리합니다.
- 최대 인원을 8명으로 확장했습니다.
- 술래는 1명, 나머지 참가자는 도망자로 배정됩니다.
- 게임 화면에 남은 시간 표시를 추가했습니다.

# BobArtist v0.0.37

## Circle Render Engine
- 술래 화면에서는 별도 원판/base circle을 그리지 않도록 변경했습니다.
- 도망자 paintCanvas를 기본 흰색 배경으로 초기화해, 아무것도 칠하지 않아도 제출 이미지 자체가 흰 원으로 보이게 했습니다.
- 원 테두리/점선/halo가 생기는 렌더링 경로를 줄였습니다.

# BobArtist v0.0.36

## 수정
- 술래 화면에서 v0.0.35의 페더 마스크 때문에 생기던 흰 링/테두리 현상을 제거했습니다.
- 술래 REVEAL/FIND 화면에서는 흰 원판과 내부 그림만 그리고, 외곽선/점선/선택선을 그리지 않도록 고정했습니다.
- 도망자/결과 화면의 기존 가이드 표시 흐름은 유지했습니다.
